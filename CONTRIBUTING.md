# Contributing to Stillness

Thanks for taking the time to contribute! This is a small, focused app — contributions that keep it minimal and purposeful are most welcome.

## Ways to contribute

- 🐛 **Bug reports** — open an Issue with steps to reproduce
- 💡 **Feature ideas** — open an Issue tagged `enhancement` before building
- 🔧 **Code PRs** — fork, branch, PR (see below)
- 🌍 **Translations** — open an Issue to discuss

## Pull Request process

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Open a PR against `main` with a clear description
5. Add co-author credits if you collaborated:
   ```
   Co-authored-by: Name <email@example.com>
   ```

## Code style

- Vanilla HTML/CSS/JS — no build tools, no frameworks
- Keep the file count low (ideally one `index.html`)
- Prefer CSS transitions over JS animations
- Mobile-first, test on real devices

## Design principles

- **Calm over clever** — this is a breathing app, not a showcase
- **No dependencies** — Google Fonts is the only external resource
- **Offline-friendly** — nothing should require a network after first load

## Reporting bugs

Please include:
- Browser and OS
- Steps to reproduce
- Expected vs actual behaviour
- Screenshot if relevant
