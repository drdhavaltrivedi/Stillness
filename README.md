## What is box breathing?

Box breathing (4-4-4-4) is a technique used by Navy SEALs, athletes, and therapists to reduce stress and sharpen focus. Each side of the "box" is one breath phase:

```
Inhale → Hold → Exhale → Hold
  4s       4s      4s      4s
```

## Features

- 🌬️ **Animated breathing guide** — dot traces the box, orb expands and contracts
- ⏱️ **Adjustable pace** — 3s, 4s, 5s, 6s, or 7s per phase
- 🔊 **Sound & haptic feedback** — optional audio tones and vibration
- 🌀 **Multiple patterns** — Box (4-4-4-4), 4-7-8, Resonance (5-5)
- 📊 **Session stats** — cycles completed and total time
- 📱 **Mobile-first** — works perfectly on any device, offline-capable

## Breathing Patterns

| Pattern | Ratio | Best for |
|---|---|---|
| Box | 4-4-4-4 | General calm, focus |
| 4-7-8 | Inhale 4, Hold 7, Exhale 8 | Sleep, anxiety relief |
| Resonance | 5-5-0-0 | Heart rate variability |
